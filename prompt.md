# BlockBot:Factory 视觉生成 Prompt

## 统一美术风格

卡通冒险游戏风格，面向中国儿童，温暖但带一点废弃工厂的神秘感。画面清晰、干净、角色轮廓明确，适合做游戏立绘和界面素材。色彩不要过暗，使用锈红、工业黄、冷却液青绿色、旧金属灰作为辅助色。避免恐怖、血腥、压迫感过强。背景可以有废弃自动化工厂、管线、警示灯、旧屏幕、散落零件，但角色必须是画面主体。透明背景，半身或全身立绘，2D digital illustration, clean lineart, high readability, game character asset.
